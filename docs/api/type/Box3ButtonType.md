玩家按下的按钮类型

|    属性     |    值     |        说明         |
| :---------: | :-------: | :-----------------: |
|    WALK     |  'walk'   |      步行按钮       |
|     RUN     |   'run'   |      奔跑按钮       |
|   CROUCH    | 'crouch'  |      下蹲按钮       |
|    JUMP     |  'jump'   |      跳跃按钮       |
| DOUBLE_JUMP |  'jump2'  |     二段跳按钮      |
|     FLY     |   'fly'   |      飞行按钮       |
|   ACTION0   | 'action0' | 鼠标左键/虚拟按钮 A |
|   ACTION1   | 'action1' | 鼠标右键/虚拟按钮 B |
